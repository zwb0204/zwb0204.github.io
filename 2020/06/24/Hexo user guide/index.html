USER Guide

  • 更新个人信息，在根目录的_config.yml 中更新博客名字，作者，git 仓库地址等等。

  • 将写好的文章直接放在目录：source/_posts下面

  • 使用命令：

    • hexo g // 如果有变动，执行命令生成
    • hexo s // 本地预览命令
    • hexo server -p 5000 // 如果4000端口被占用，则使用此命令更换端口
    • hexo d // commit 本地改动到github
